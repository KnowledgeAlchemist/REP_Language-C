# Instruções do projeto

``` yaml
3) Faça um programa que receba o preço de um produto, calcule e mostre o novo preço,sabendo-se que este sofreu um desconto de 10%.
```

``` c
#include <stdio.h>

int main() {
  int priceProduct, priceProductDesconto;

  printf("Digite o valor do produto: ");
  scanf("%d", &priceProduct);

  priceProductDesconto = priceProduct - (priceProduct * 0.10);

  printf("O valor do produto com o desconto é: %d", priceProductDesconto);
}
```
