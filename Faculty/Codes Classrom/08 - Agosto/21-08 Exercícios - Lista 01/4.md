# Instruções do projeto

``` yaml
4) Faça um programa que receba o custo de um espetáculo teatral e o preço do convite desse espetáculo. Esse programa deve calcular e mostrar a quantidade de convites que devem ser vendidos para que pelo menos o custo do espetáculo seja alcançado.
```

``` c
#include <stdio.h>

int main() {
  int custoEspetaculo, precoConvite;

  printf("Digite o custo do espetaculo: ")
  scanf("%d", &custoEspetaculo)

  printf("Insira o valor do convite: ");
  scanf("%d", &precoConvite);

  quantidadeConvites = custoEspetaculo / precoConvite;

  printf("Serao necessários %d convites para custear o espetáculo", quantidadeConvites)
}
```
